# Trabalho 02 - ACO para TSP
Este projeto implementa o algoritmo de Colônia de Formigas para resolver o Problema do Caixeiro Viajante.

## Como Rodar
1. Instale as dependências: `pip install numpy matplotlib`
2. Execute: `python src/main.py`
3. Os resultados estarão na pasta `output/`.
